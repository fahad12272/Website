What is the time complexity of searching for an element in a balanced binary search tree of n nodes?

Choose one answer
Explanation

In a balanced BST (e.g. AVL or Red-Black tree), the height is bounded by O(log n). Search descends the tree from root to leaf, comparing once per level — yielding O(log n) lookups.
